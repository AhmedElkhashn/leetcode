# Daily Temperature 

**Difficulty:** medium  
**Topic:** Stack  
**Link:** [LeetCode](https://leetcode.com/problems/daily-temperatures/)


## Problem Statement

Given an array of integers temperatures represents the daily temperatures, return an array answer such that answer[i] is the number of days you have to wait after the ith day to get a warmer temperature. If there is no future day for which this is possible, keep answer[i] == 0 instead.


## Approach

Iterate through the temperatures and for each one you check if it's warmer than what is on top of the stack, if it is we pop the stack and append the answer with is the current index - the index stored in the stack. Keep popping and resolving until the stack is empty or the current temperature is no longer warmer than the top. 


## Solution

```python
from typing import List


class Solution:
    def dailyTemperatures(self, temperatures: List[int]) -> List[int]:
        output = [0] * len(temperatures)
        stack = []


        for i in range(len(temperatures)):
           
            while stack and temperatures[i] > stack[-1][0]:
                temp, j = stack.pop()
                output[j] = i -j
           
            stack.append([temperatures[i],i])


        return output
```
