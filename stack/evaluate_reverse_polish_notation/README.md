# Evaluate Reverse Polish Notation

**Difficulty:** medium  
**Topic:** Stack  
**Link:** [LeetCode](https://leetcode.com/problems/evaluate-reverse-polish-notation/)


## Problem Statement

You are given an array of strings tokens that represents an arithmetic expression in Reverse Polish Notation. Evaluate the expression and return an integer that represents its value.

Note that:

- The valid operators are '+', '-', '*', and '/'.
- Each operand may be an integer or another expression.
- The division between two integers always truncates toward zero.
- There will not be any division by zero.
- The input represents a valid arithmetic expression in Reverse Polish Notation.
- The answer and all intermediate calculations can be represented in a 32-bit integer.


## Approach

Loop through the list if it's a number we push it to the stack otherwise if it's an operation we pop the last two values in the stack and perform the operation then push the result. Then return the last values in the stack. 


## Solution

```python
from typing import List


class Solution:
    def evalRPN(self, tokens: List[str]) -> int:
        stack = []


        for n in tokens:
            if n == "+":
                val1 = stack.pop()
                val2 = stack.pop()
                stack.append(val1+val2)
            elif n == "-":
                val1 = stack.pop()
                val2 = stack.pop()
                stack.append(val2-val1)
            elif n == "*":
                val1 = stack.pop()
                val2 = stack.pop()
                stack.append(val1*val2)
            elif n == "/":
                val1 = stack.pop()
                val2 = stack.pop()
                stack.append(int(val2/val1))
            else:
                stack.append(int(n))


        return stack[0]
```
