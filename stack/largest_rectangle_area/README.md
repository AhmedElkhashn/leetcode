# Largest Rectangle in Histogram

**Difficulty:** hard  
**Topic:** Stack  
**Link:** [LeetCode](https://leetcode.com/problems/largest-rectangle-in-histogram/)


## Problem Statement

Given an array of integers heights representing the histogram's bar height where the width of each bar is 1, return the area of the largest rectangle in the histogram.


## Approach

We iterate through the loop and we check if the height is shorter than the previous one (top of the stack) if it is that means we cant extend the rectangle to the right so we pop it from the stack. we calculate the are of that rectangle and store it if its the max. we then append the new bar or rectangle to the stack. A rectangle could be formed if we popped from the stack this rectangle is formed from the current height extending to the left. we then loop through the stack which contains the rectangles that made it to the end of the histogram calculate their area and check if its the max. Finally we return the max area.


## Solution

```python
from typing import List


class Solution:
    def largestRectangleArea(self, heights: List[int]) -> int:
        
        stack = []

        area = 0

        for i in range(len(heights)):
            
            index = i

            while stack and heights[i] < stack[-1][1]:                
                j,h = stack.pop()
                area = max(area, h * (i - j))
                index = j

            stack.append([index,heights[i]])

    
        for i in range(len(stack)):
            area = max(area, stack[i][1] * (len(heights) - stack[i][0]))

        return area
```
