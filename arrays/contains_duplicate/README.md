# Contains Duplicate

**Difficulty:** easy  
**Topic:** Arrays  
**Link:** [LeetCode](https://leetcode.com/problems/contains-duplicate/)


## Problem Statement

Given an integer array nums, return true if any value appears at least twice in the array, and return false if every element is distinct.


## Approach

Use a hash set to track numbers we've already seen as we iterate. If we encounter a number that's already in the set, we immediately return True. If we finish the loop without a match, return False. A set is used because the lookup is O(1)


## Solution

```python
from typing import List

class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        numbers = set()
        for num in nums:
            if num in numbers:
                return True
            numbers.add(num)
        return False
```
