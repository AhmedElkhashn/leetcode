# Group Anagrams

**Difficulty:** medium  
**Topic:** Hashing  
**Link:** [LeetCode](https://leetcode.com/problems/group-anagrams/)


## Problem Statement

Given an array of strings strs, group the anagrams together. You can return the answer in any order.


## Approach

For each string we create a list with the count of the letters and then convert it to a tuple so it can be used as a key. All strings that are anagrams of each other will produce the exact same tuple, so they all get grouped under the same key in the hash map, the values are then put into a list and returned.


## Solution

```python
from collections import defaultdict
from typing import List


class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        hashmap = defaultdict(list)

        for s in strs:
            count = [0] * 26      

            for c in s:

                count[ord(c) - ord("a")] += 1

            hashmap[tuple(count)].append(s)

        return list(hashmap.values())

```
