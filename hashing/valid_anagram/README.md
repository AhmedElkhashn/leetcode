# Valid Anagram

**Difficulty:** easy  
**Topic:** Hashing  
**Link:** [LeetCode](https://leetcode.com/problems/valid-anagram/)


## Problem Statement

Given two strings s and t, return true if t is an anagram of s, and false otherwise.


## Approach

First check if they have the same length, if they do they can't be anagrams. Create a hash table for each string with the count of each character and compare the two tables.


## Solution

```python
class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
       
        if len(s) != len(t):
            return False


        countS = {}
        countT = {}


        for  i in range(len(s)):
            countS[s[i]] = 1 + countS.get(s[i],0)
            countT[t[i]] = 1 + countT.get(t[i],0)
           
        for c in countS:
            if countS[c] != countT.get(c,0):
                return False
       
        return True
```
